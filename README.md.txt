The Text Tools App is designed to offer convenient text-related operations. It provides features like text-to-speech conversion, text-to-PDF conversion, and Word-to-PDF conversion. The app is built using Streamlit, gTTS (Google Text-to-Speech), FPDF, and docx2pdf libraries.

Features

1. Text to Speech Conversion:
    - Enter text and convert it to speech in various languages.
    - Control the speed of the generated speech.

2. Text to PDF Conversion:
    - Convert entered text into a downloadable PDF file.

3. Word to PDF Conversion:
    - Upload a Word document and convert it to a PDF file.
